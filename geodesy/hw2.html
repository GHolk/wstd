### 擾動位能
在任一點上， **實際重力位** 與 **正常重力位** 的差。
實際重力位就是現場測得的重力位能；
而正常重力位則是用大地橢球推算出來的。

擾動位能滿足 Laplace Equation ，

Δ W = Δ V + Δ Φ = 0 + 2 ω 2

Δ U = Δ V E +
Δ Φ E = 0 + 2 ω 2

由於橢球自轉速率與實際地球自轉速率幾乎相同，
Φ E ≒ Φ 。

且正常引力位與引力位也滿足
Δ V = Δ V E = 0

故
Δ T = Δ ( W - U ) =
Δ ( V + Φ - V E - ΦE ) = 
Δ V - Δ V E =
0 - 0 = 0

    .EQ
    T ( r , theta , lambda ) =
    Sum from n = 0 to infinity
    ( R over r ) sup { n + 1 }
    T sub n ( theta , lambda )
    .EN

T(r,θ,λ)=Sumn=0infinity(Rr)n+1Tn(θ,&lamda;)

擾動位為實際重力位與正常重力位的相減，
各階的 cos sin 系數也是。
由於正常重力位為橢球，沒有經度方向的變化，
若 n 為偶數，正常重力位 cos 系數為 0 ， CTnm = Cnm 。

若橢球與實際地球質量相等，則 0 階項為 0 。
若橢球中心與實際地球中心重合，則 1 階項為 0 。


### 重力擾動
重力擾動則是加速度，
也就是一點上 **實際重力加速度** 與
**正常重力加速度** 的差，
是一個向量，一般直接表示為向量的長度。

    .EQ
    delta vec g = vec { g sub p } - vec { gamma sub p }
    .EN

δ→g=→gp−→γp

一般重力擾動的計算較困難，
因為要計算不在橢球上的重力加速度，
再與實際測得的重力加速度相減。
也可以簡單以重力異常近似。

### 重力異常
重力異常是橢球上一點 Q 的正常重力加速度，
與該點垂線與地表交點 P 的實際重力加速度差，
也是一個向量，因為角度不會差太多，
所以常表示為純量。

.EQ
Delta vec g = vec g sub P - vec gamma sub Q
.EN

可以用重力加速度儀在地面測得 g P ，
再用大地橢球算得 γ Q ，
相減即是重力異常。

### 大地起伏
**大地水準面** 與 **大地橢球** 間的 **橢球高** 差，
在地表上一點的話，大地起伏就是正高減去橢球高， N = H - h 。

### 垂線偏差
重力線與橢球法線方向的角度差。
與 **平均地球橢球** 的稱為 **絕對垂線偏差** ，
與各參考橢球的稱為 **相對垂線偏差** 。

### 高程異常
**橢球高** 與 **正常高** 差，
也就是 **似地表面** 與地表面的差。

沿橢球面法線方向，
重力位與地表上實際一點相同的點，
該點與橢球面距離稱為正常高。
計算得與所有地表上的點位能相同的點
所構成的平面，則稱為似地表面。

此公式將觀測得的地表實際重力，

.EN

這樣只要有所有 geoid 上的 Δg ，就能解 T 。

無限寛廣的海洋
--------------
假設你於一個無限寬廣的海水面上測量重力，假設儀器分別位於船上和固定柱上，儀器到海水面距離為零，船上和固定柱上儀器至水面距離為零，此時水面突然上升2公尺，請問此時重力儀上的重力讀數分別變化多少? (記得單位和正負號)?
需解釋變化原因及並列出計算過程

因為水面上升了 2m ，我們離原始海水遠了 2m ，
可以視為反向的自由空間約化。

F ≈ 0.3086 H = - 0.6172 mgals

g' = g + F = g - 0.6172

此時我們是懸在空中，離海面 2m ，
還得要把這 2m 填滿海水，
可以視為反向的布格平板改正。

A = 2 π G ρ H = 2 π × 0.0066732 H = 0.0419101123

g'' = g' + A = g - 0.6172 + 0.041910 = g + 0.659110

蠻有趣的，海水面上升，雖然質量增加，
但因為離地球更遠，增加質量不比遠離地球的效益，
重力反而減小了。

重力約化
--------
請詳細說明如何求得Free-air anomaly、Bouguer anomaly、Faye anomaly 、Isostatic anomaly(各種重力約化需說明)。若一點位(緯度：18.543度; 經度:118.554度，正高20.321公尺)之觀測絕對重力值為9.8232542 m/s^2，請問該點在大地水準面對應點之Free-air anomaly和Bouguer anomaly為何?

 1. 中文名詞定義，說明各約化過程並列式
 2. 需列出計算過程


### free-air anomaly
自由空間約化，將近地表一點在垂直方向移動，
靠近或遠離地心 H 後重力的變化。
且這個移動不能造成質量從點的上方移動下方，或反之；
可以想像成點是在空中移動。

由於移動的尺度相對離地球質心的距離很小，
可以視為重力梯度不變，也就是可以線性近似。

.EQ
F ~=
{ part gamma } over { part H } H ~=
0.3086 H mgals
.EN

g' = g - F = 9.8232542 - 0.3086 * 20.321 / 1000 = 9.8169831394

### Bouguer anomaly
布格平板改正，將近一點以下厚度 H 的均勻質量柱體移除，
並將點垂直移動 -H 。
布格平板是很扁的圓柱體，像一塊平板，
半徑 a 高度 b 密度 ρ 。

.EQ
A sub 0 = 2 pi G rho
( a + b -
sqrt { a sup 2 + b sup 2 } )
.EN

若平板為無限寛，（和地表一樣寬。）
則可以取極限成：

.EQ
lim from { a -> infinity }
2 pi G rho ( a + b -
sqrt { a sup 2 + b sup 2 } ) =
2 pi G rho b
.EN

近地表地殼密度約為 ρ = 2.67 g/cm³ ，
帶入後 A = 0.1119 H ，
只改正 A 稱為不完全布格約化。
再加上自由空間約化，
將點移到除去平板後的位置才是完全布格

A = 0.1119 H > 0.3086 H = F ，
可以發現移除後少了這一塊平板造成的變化，
小於移動 -H 後的影響量，
所以布格改正後重力其實是變大的。

g' = g - F + A = g - 0.0062710606 + 0.0022739199 = 9.8192570593


### Faye 約化
因為布格改正會移除太多質量，
讓 geoid 移動過大。
所以只做自由空間約化與地型改正，
稱為 Faya 約化。

地型改正則是把凹的填起來，
把凸的削平，讓地表成為一個平面。

### Isostatic anomaly
均衡約化是把 geoid 下的密度約化到一樣。
這樣做布格約化時，就不會發生
在不同密度減去相同質量的情形。

大地水準面約化
--------------
若地表 P 點量測重力為g，請問大地水準面上 P0 點重力為何？
P至P0點的平均重力為何？需列出計算過程及答案

若 P 與 P0 正高差 H ，
計算 P0 的 g 需先將 P 做自由空間約化
移動 H 到 P0 上：

F = 0.3086 H

然後原本在下方的質量移到了上方，
所以要減去兩倍的布格平板：

A = 2 × 0.1119 H

g' = g + F - A

平均重力則是簡單兩點的平均：

(g + g') / 2 = g + 0.0424 H
